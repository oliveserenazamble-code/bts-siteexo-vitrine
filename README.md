LES COMMANDES GIT UTILISÉES

git clone : qui permet de copier un projet distant en local
git checkout : qui permet d'aller sur une branche
git checkout -b : qui permet de créer de nouvelle branche
git status : qui permet de voir la situations de tes fichiers
git add . : qui permet d'ajouter ou enregistrer tous les fichiers qui seront commité
git add: qui permet d'ajouter au plus fichier pour son commit
git commit -m "": qui permet de sauvegarder ou d'enregistrer les modifications
git push : qui permet de pousser tout ce que tu as eu à faire ou un projet  sur git dans git hub dans un nouveau repository
git pull : mettre à jour un projet déjà cloné
